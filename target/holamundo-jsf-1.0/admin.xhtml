DIAGRAMAS CASOS DE USO

PRODUCTOS:

Este caso de uso tiene como condición que el usuario se encuentre autenticado como un administrador. Desde la opción de administrar productos el admin tiene la posibilidad de gestionar un producto con las opciones de registrar modificar eliminar y consultar. A la hora de realizar el registro de un producto nuevo en el sistema, se le solicitara ID del producto, nombre, precio, cantidad y tipo.

TIENDAS

Este caso de uso tiene como condición que el usuario este autenticado como administrador. Desde la opción de administrar tiendas, el admin tiene la posibilidad de gestionar las tiendas con las opciones de registrar, modificar, eliminar y consultar. A la hora de registrar una tienda nueva, el sistema le solicitara ID tienda, ID del municipio y dirección de la tienda.

CATEGORIAS

La aplicación le permitirá al usuario que este autenticado como administrador que realice gestión de categorías como registrar categoría, modificar, eliminar y consultar. A la hora de realizar el registro de una nueva categoría la aplicación le solicitara al administrador ingresar ID categoría y el nombre de la categoría.

PEDIDOS

Este caso de uso tiene como condición que el usuario se encuentre autenticado como un administrador. Desde la opción de gestionar pedido el admin tiene la posibilidad de gestionar un pedido con las opciones de registrar modificar eliminar y consultar. A la hora de realizar el registro de un pedido nuevo en el sistema, se le solicitara ID del pedido, ID tienda, fecha y hora y precio total.

EMPLEADOS

La aplicación le permitirá al usuario que este autenticado como administrador que realice gestión de empleados como registrar, modificar, eliminar y consultar. A la hora de realizar el registro de un nuevo empleado la aplicación le solicitara al administrador ingresar ID empleado, ID tienda, ID municipio, apellido y nombre del empleado.

MUNICIPIOS

Desde la opción de gestionar municipios el admin tiene la posibilidad de gestionar un municipio con las opciones de registrar modificar eliminar y consultar. A la hora de realizar el registro de un municipio nuevo en el sistema, se le solicitara ID municipio, ID departamento, nombre.

DEPARTAMENTOS

Desde la opción de gestionar departamentos el admin tiene la posibilidad de gestionar un departamento con las opciones de registrar modificar eliminar y consultar. A la hora de realizar el registro de un departamento nuevo en el sistema, se le solicitara ID departamento, nombre.
